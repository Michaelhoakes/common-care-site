# Hero font — Neue Haas Grotesk Display Pro

Place these `.woff2` files here (names must match exactly, or update `globals.css` @font-face `src`):

| File | Use | CSS weight |
|------|-----|------------|
| `NeueHaasDisplayPro-35ExtraLight.woff2` | Hero h1 (thin) | 250 |
| `NeueHaasDisplayPro-55Roman.woff2` | 55 Roman / Regular | 400 |
| `NeueHaasDisplayPro-65Medium.woff2` | Hero subtext (medium) | 500 |

If your export uses different names (e.g. `Neue Haas Grotesk Display Pro 55 Roman.woff2`), rename to the table above or change the `url(...)` in `src/app/globals.css` to match your filename.
